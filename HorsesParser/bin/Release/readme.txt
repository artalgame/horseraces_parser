Обновление данных:
	для обновления данных используются две кнопки:
	1) Обновление - подгружает новые данные, которых еще нет в 	   системе. Т.е. свежие дни. Парсит также BPS и Place
	2) Полное обновение - начинает полное обновление данных, 	   даже если они есть в системе - реккомендуется только в 	   случае, когда таблицы с данными потеряны.

Также при выборе нужной даты в списке Даты программа попытается скачать данные для тех гонок, у которых не спаршен BSP и Place. Иногда придется подождать, чтобы данные спарсились - если данные не парсятся(Например отсутствует подключение к Интернету - тогда в колонках BSP и Place будут 0)

При запуске программы - данные загружаются в память - процесс небыстрый, поэтому нужно подождать минуту-две.

При выключении программы данные пересохраняются - это также занимает несколько десятков секунд.
